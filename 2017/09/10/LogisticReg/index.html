Logistic Regression 的相关数学推导

发表于 2017-09-10 |

Coursera机器学习课程（by Andrew Ng）的学习笔记

对Logistic Regression课程中的数学推导进行补充记录，主要内容如下：

  • 利用极大似然估计推导Cost Function
  • 如何理解从模型函数到概率的转换
  • 梯度的数学推导

极大似然估计到Cost Function

在这之前，我们先看看线性回归的问题。我们都会学到最小二乘法，使得训练集中残差$\epsilon^{\left(i\right)}$的平方和达到最小，而使得模型在整个训练集的整体误差达到最小。也即是线性回归模型的Cost Function如课程中所示：

我会有这样的疑问为什么一定是平方和。因为很多函数构造都可以达到这个目的。当然Andrew在课程中给出了一个回答是，这个函数是凸函数，在最优化问题中没有局部最优，只有全局最优（简单的说就是好用）。到了Logistic Regression课程中，Andrew直接给出了如下更复杂的Cost Function：

有没有一个统一的方法将这两种模型的Cost Function统一起来。答案肯定的，也就是极大似然估计（Maximum Likelihood Estimate，MLE）的方法（在Andrew的课程中提到）。这一节对极大似然估计进行一些探讨和解析。

极大似然估计方法的定义摘自[ Wikipedia: Maximum Likelihood Estimation ]。极大似然估计方法主要应用在针对模型参数的估计中。给定任何一种模型$y = {\mathop{\rm h}\nolimits}_{\Theta} \left( x \right)$，其模型参数$\Theta$的取值有无数种可能。而拟合的目的是通过一些列观察值$\left(x^{\left(i\right)}, y^{\left(i\right)} \right)$，也即是训练集，选择出最合理的模型参数$\Theta$。什么是最合理的，如何去量化这个概念，极大似然估计能回答我们这个问题。如果我们认为输出值$y$是随机变量，那么对任意一个输入输出对$\left(x^{\left(i\right)}, y^{\left(i\right)} \right)$，都有如下的条件概率描述：在输入$x^{\left(i\right)}$的情况下观察到输出为$y^{\left(i\right)}$的概率为${\rm{P}}\left( {y^{\left( i \right)}}\left| {x^{\left( i \right)}};\Theta \right. \right)$。而极大似然估计就是找到模型参数$\Theta$使得整个训练集$\left(x^{\left(i\right)}, y^{\left(i\right)} \right)$出现的条件概率最大。数学描述如下所示：
$$\Theta = \mathop {\arg \max }\limits_\Theta L = \mathop {\arg \max }\limits_\Theta \prod\limits_i {\rm{P}}\left( {{y^{\left( i \right)}}\left| {{x^{\left( i \right)}};\Theta } \right.} \right)$$

  • 线性回归

    在线性回归中，我们认为残差服从正态分布$y^{\left(i\right)} - {\mathop{\rm h}\nolimits} \left( x;\Theta \right) = \epsilon^{\left(i\right)} \sim N\left(0, \sigma \right)$，于是有每个参数点的条件概率如下：

    $${\rm{P}}\left( {y^{\left( i \right)}}\left| {x^{\left( i \right)}};\Theta \right. \right) = \frac{1}{{\sqrt {2\pi } \sigma }}{e^{ - \frac{{{{\left( {{{\mathop{\rm h}\nolimits}_\Theta }\left( {{x^{\left( i \right)}}} \right) - {y^{\left( i \right)}}} \right)}^2}}}{{2{\sigma ^2}}}}}$$

    于是对于训练集的条件概率为$L=C{e^{ - \frac{{\sum\limits_{i = 1}^m {{{\left( {{{\mathop{\rm h}\nolimits}_\Theta }\left( {{x^{\left( i \right)}}} \right) - {y^{\left( i \right)}}} \right)}^2}} }}{{2{\sigma ^2}}}}}$，其中$C$是我们不太关注的常数项。而极大似然估计就是求得使上述概率最大的模型参数$\Theta$。通常情况下，会对上述概率求$\log$，转换为如下：

    $$\begin{array}{l} \Theta = \mathop {\arg \max }\limits_\Theta L = \mathop {\arg \max }\limits_\Theta \left( {\log L} \right) = \\ \;\;\;\,= \mathop {\arg \max }\limits_\Theta \left( { - \frac{{\sum\limits_{i = 1}^m {{{\left( {{{\mathop{\rm h}\nolimits}_\Theta }\left( {{x^{\left( i \right)}}} \right) - {y^{\left( i \right)}}} \right)}^2}} }}{{2{\sigma ^2}}}} \right)\\ \;\;\;\,= \mathop {\arg \min }\limits_\Theta \left( {\sum\limits_{i = 1}^m {{{\left( {{{\mathop{\rm h}\nolimits}_\Theta }\left( {{x^{\left( i \right)}}} \right) - {y^{\left( i \right)}}} \right)}^2}} } \right) \end{array}$$

    上式与之前的Cost Function形式完全一致。

  • Logistic Regression

    进一步在Logistic Regression中，由于输出值只取0-1二值，所以容易想到输出值服从伯努利分布。于是有如下条件概率：

    $$\begin{array}{l} {\rm{P}}\left( {{y^{\left( i \right)}}\left| {{x^{\left( i \right)}};\Theta } \right.} \right) = {\rm{P}}{\left( {{y^{\left( i \right)}} = 1\left| {{x^{\left( i \right)}};\Theta } \right.} \right)^{{y^{\left( i \right)}}}}{\left( {1 - {\rm{P}}\left( {{y^{\left( i \right)}} = 1\left| {{x^{\left( i \right)}};\Theta } \right.} \right)} \right)^{1 - {y^{\left( i \right)}}}}\\ {\kern 51.5pt}= {{\mathop{\rm h}\nolimits}_\Theta }{\left( {{x^{\left( i \right)}}} \right)^{{y^{\left( i \right)}}}}{\left( {1 - {{\mathop{\rm h}\nolimits}_\Theta }\left( {{x^{\left( i \right)}}} \right)} \right)^{1 - {y^{\left( i \right)}}}} \end{array}$$

    同样的，求解极大似然估计的过程如下：

    $$\begin{array}{l} \Theta = \mathop {\arg \max }\limits_\Theta L = \mathop {\arg \max }\limits_\Theta \left( {\log L} \right) = \\ {\kern 6.5pt}= \mathop {\arg \max }\limits_\Theta \left( {\sum\limits_{i = 1}^m {\left( {{y^{\left( i \right)}}\log \left( {{\mathop{\rm h}\nolimits}_{\Theta}\left( {{x^{\left( i \right)}}} \right)} \right) + \left( {1 - {y^{\left( i \right)}}} \right)\log \left( {1 - {\mathop{\rm h}\nolimits}_{\Theta}\left( {{x^{\left( i \right)}}} \right)} \right)} \right)} } \right)\\ {\kern 6.5pt}= \mathop {\arg \min }\limits_\Theta \left( { - \sum\limits_{i = 1}^m {\left( {{y^{\left( i \right)}}\log \left( {{\mathop{\rm h}\nolimits}_{\Theta}\left( {{x^{\left( i \right)}}} \right)} \right) + \left( {1 - {y^{\left( i \right)}}} \right)\log \left( {1 - {\mathop{\rm h}\nolimits}_{\Theta}\left( {{x^{\left( i \right)}}} \right)} \right)} \right)} } \right) \end{array}$$

    上式同样与之前的Cost Function形式完全一致。

在上一节中对Logistic Regression模型的极大似然估计过程的推导中，我们直接使用了一个等式${\rm{P}}{\left( {{y^{\left( i \right)}} = 1\left| {{x^{\left( i \right)}};\Theta } \right.} \right)} = {{\mathop{\rm h}\nolimits}_\Theta }\left( {{x^{\left( i \right)}}} \right)$

Cost Function的梯度推导

这也是Andrew的课程中没有涉及到的内容。在网络上有一些博客专门做了详细的推导，但是我想从Matrix Calculus出发做更优美的推导。

首先将Cost Function进行向量化，得到如下：
$${\mathop{\rm J}\nolimits} \left( \Theta \right) = - \frac{1}{m}\left( {{{\bf{y}}^{\sf{T}}}\log \left( {{\mathop{\rm g}\nolimits} \left( {{\bf{X}}\Theta } \right)} \right) + {{\left( {1 - {\bf{y}}} \right)}^{\sf{T}}}\log \left( {1 - {\mathop{\rm g}\nolimits} \left( {{\bf{X}}\Theta } \right)} \right)} \right)$$

其中${\mathop{\rm g}\nolimits}\left(x\right)=\frac{1}{1+e^{-x}}$，而其导函数${\mathop{\rm g}\nolimits}’\left(x\right)=\frac{e^{-x}}{\left(1+e^{-x}\right)^2}={\mathop{\rm g}\nolimits}\left(x\right)\left(1-{\mathop{\rm g}\nolimits}\left(x\right)\right)$

其中${\bf{X}} = \left[ \begin{array}{l} {x^{{{\left( 1 \right)}^{\sf{T}}}}}\\ {x^{{{\left( 1 \right)}^{\sf{T}}}}}\\ ...\\ {x^{{{\left( m \right)}^{\sf{T}}}}} \end{array} \right]_{m\times n}$，${\bf{y}}=\left[ \begin{array}{l} {y^{{{\left( 1 \right)}}}}\\ {y^{{{\left( 1 \right)}}}}\\ ...\\ {y^{{{\left( m \right)}}}} \end{array} \right]_{m\times 1}$。

于是有以下求解梯度的过程：

$$\begin{array}{l} \frac{{\partial {\mathop{\rm J}\nolimits} \left( \Theta \right)}}{{\partial \Theta }} = - \frac{1}{m}\left( {{{\bf{X}}^\sf{T}}{\mathop{\rm g}\nolimits}'\left( {{\bf{X}}\Theta } \right)\frac{1}{{{\mathop{\rm g}\nolimits} \left( {{\bf{X}}\Theta } \right)}}{\bf{y}} + {{\bf{X}}^\sf{T}}\left( { - {\mathop{\rm g}\nolimits}'\left( {{\bf{X}}\Theta } \right)} \right)\frac{1}{{1 - {\mathop{\rm g}\nolimits} \left( {{\bf{X}}\Theta } \right)}}\left( {1 - {\bf{y}}} \right)} \right)\\ = - \frac{1}{m}\left( {{{\bf{X}}^\sf{T}}\left( {1 - {\mathop{\rm g}\nolimits} \left( {{\bf{X}}\Theta } \right)} \right){\bf{y}} - {{\bf{X}}^\sf{T}}{\mathop{\rm g}\nolimits} \left( {{\bf{X}}\Theta } \right)\left( {1 - {\bf{y}}} \right)} \right)\\ = - \frac{1}{m}\left( {{{\bf{X}}^\sf{T}}{\bf{y}} - {{\bf{X}}^\sf{T}}{\mathop{\rm g}\nolimits} \left( {{\bf{X}}\Theta } \right)} \right)\\ = \frac{1}{m}{{\bf{X}}^\sf{T}}\left( {{{\mathop{\rm h}\nolimits} _\Theta }\left( {\bf{X}} \right) - {\bf{y}}} \right) \end{array}$$
